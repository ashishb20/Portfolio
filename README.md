# Portfolio

A modern, responsive personal portfolio website.

## features
- **Dark/light Mode**: User-toggleable theme.
- **Smooth Animation**: Utilizing `framer-motion` for engaging entry animation and transitions.
- **Github INtegration**: Displays real-time Github statistics and contribution streaks.
- **LeetCode Integeration**: Visualizes LeetCode problem-solving statistics with a heatmap.
- **Project Showcase**: Dedicated section to display projects with descriptions and links.
- **Contact Form**: Integrated contact section.


## Installation
1. **Clone the repository**
```bash
git clone https://github.com/ashishb20/portfolio.git
cd portfolio
```
2. **Install dependencies**

```bash
npm install
```

3.**Start the development server**
```bash
npm run dev
```
The application will act at `http://localhost:5173` (or the port shown in your terminal).

##  Customization
### Updating Personal Information
- **Hero Section** : Edit `src/components/Hero.jsx` to update our name, title, and bio.
- **Projects**: update `src/components/Projects.jsx` with your own project data.
- **Contributions** : update `src/components/Contribution.jsx` to replace `ashishb20` with your Github and LeetCode usernames.
